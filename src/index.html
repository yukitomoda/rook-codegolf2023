コードゴルフって？

コードゴルフとは、あるお題のプログラムを
できるだけ短い文字数で作成する遊びです。

サンプル問題を見てみよう

実際に、どのように短いコードを作成するか見てみましょう
※この問題はサンプル用の問題です。
サンプル

問題

コンソールに1から順番に整数を100行出力してください。
ただし、その整数が3の倍数である場合は整数の代わりに “Fizz” を、5の倍数である場合は整数の代わりに “Buzz” を出力してください。
また、3の倍数かつ5の倍数（つまり、15の倍数）である場合には “Fizzbuzz” を出力してください。
サンプル

実装例

for (let i = 1; i <= 100; i++) {
　if (i % 15 == 0) {
　　console.log('FizzBuzz');
　} else if (i % 3 == 0) {
　　console.log('Fizz');
　} else if (i % 5 == 0) {
　　console.log('Buzz');
　} else {
　　console.log(i);
　}
}

この問題でコードゴルフをすると・・・
サンプル

コード

for (let i = 1; i <= 100; i++) {
　if (i % 15 == 0) {
　　console.log('FizzBuzz');
　} else if (i % 3 == 0) {
　　console.log('Fizz');
　} else if (i % 5 == 0) {
　　console.log('Buzz');
　} else {
　　console.log(i);
　}
}

大会概要

応募資格 大会期間中、株式会社ルークの正社員であること
スケジュール
  • 12/4（月）問題発表・エントリー開始
  • 12/18（月）23:59 エントリー締切
  • 12/25（月） 結果発表
プログラミング言語 JavaScript （Node.js バージョン18）
ルール 問題発表時に、作成するプログラムのお題と、
そのお題の実装例コード※が発表されます。
お題のプログラムをできるだけ短いコードで作成してください。

※実装例コードは、お題のプログラムの一例であり、必ずしも正解ではありません。

応募する際、このコードを修正して短くしていくことも、一からコードを作成することもできます。

  • インターネット上のコードを参考にしたり、ツールなどを使用してプログラムを作成することは可能ですが、ソースコードの著作者はエントリーした本人でなければなりません。

  • コードの長さは、UTF-8エンコード時のバイト数でカウントします。ひらがななどを使用すると文字数よりも長くカウントされますのでご注意ください。

  • コードはNode.jsで実行します。ブラウザ固有の機能は使用できませんのでご注意ください。
