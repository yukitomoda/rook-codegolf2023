コードゴルフって?
コードゴルフとは、あるお題のプログラムを
できるだけ短い文字数で 作成する遊びです。
サンプル問題を見てみよう！
実際に、どのように短いコードを作成するか見てみましょう！
※この問題はサンプル用の問題です。
サンプル 問題
コンソールに1から順番に整数を100行出力してください。
ただし、その整数が3の倍数である場合は整数の代
わりに “Fizz” を、5の倍数である場合は整数の代わ
りに “Buzz” を出力してください。
コンソールに1から順番に整数を100行出力してください。
また、3の倍数かつ5の倍数（つまり、15の倍数）で
ある場合には “Fizzbuzz” を出力してください。
サンプル 実装例
  1. for (let i = 1; i <= 100; i++) {
  2.     if (i % 15 == 0) {
  3.         console.log( 'FizzBuzz');
  4.     } else if (i % 3 == 0) {
  5.         console.log('Fizz');
  6.     } else if (i % 5 == 0) {
  7.         console.log('Buzz');
  8.     } else {
  9.         console.log(i);
  10.     }
  11. }

プログラミング言語
JavaScript （Node.js バージョン18 ）
ルール
問題発表時に、作成するプログラムのお題と
そのお題の実装例コード※が発表されます。
お題のプログラム できるだけ短いコード で作成してください。
※：お題のプログラムを文字数の削減などすることなく簡単に実装したソースコードです。
応募する際、このコードを修正して短くしていくことも、一からコードを作成することもできます。
  1. インターネット上のコードを参考にしたり、ツールなどを使用してプログラムを作成することは可能ですが、ソースコードの著作者はエントリーした本人でなければなりません。
  2. コードの長さは、UTF-8エンコード時のバイト数でカウントします。ひらがななどを使用すると文字数よりも長くカウントされますのでご注意ください。
  3. コードはNode.jsで実行します。ブラウザ固有の機能は使用できませんのでご注意ください。

Q&A
Q.
インターネット等で類似問題を探して、そのコードを参考にしても良いですか？
A.
はい、問題ありません。
ただし、
コードの著作者はエントリーする本人
である必要があります。よって、他の人の著作物を（たとえすべての著作権が譲渡されていたとしても）あなたのコードとして提出することはできません。
Q.
コードの整形など（例えばminify）のためにツールを使用しても良いですか？
A.
はい、問題ありません。
Q.
エントリー後にコードの変更はできますか？
A.
はい、可能です。
エントリー後にコードを変更する場合は再度エントリーフォームにアクセスして、回答を編集してください。
